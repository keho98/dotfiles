# dotfiles
Personal Dotfiles

## Features
- gitignore
- sets up zsh using oh-my-zsh

## Installation

```sh
$ git clone https://github.com/nicksp/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ chmod +x setup.sh
$ ./setup.sh
```
